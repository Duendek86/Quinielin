# Quinielin

Generador de apuestas para la Quiniela de la Liga Española basado en estadísticas.

## Requisitos

- [ZenC Compiler](https://github.com/z-libs/Zen-C)

## Instalación y Uso

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Duendek86/Quinielin.git
   ```
2. Compila el proyecto:
   ```bash
   build.bat
   ```
3. Ejecuta el generador:
   ```bash
   bin/quinielin.exe
   ```

## Datos

Los datos estadísticos se obtienen de [DataHub Spanish La Liga](https://datahub.io/core/spanish-la-liga).
Se espera que los archivos `.csv` se encuentren en la carpeta `data/`.
